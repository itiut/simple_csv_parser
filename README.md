# SimpleCsvParser

Simple CSV parser based on RFC 4180

## Installation

Add this line to your application's Gemfile:

    gem 'simple_csv_parser'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simple_csv_parser

## Usage

```console
$ bundle exec bin/simple_csv_parser CSVFILE
```
